# Candidate Tracker Template

Copy this file to `local/candidate-tracker.local.md` and keep it private.

Use this Markdown table to maintain all screened candidates in ranked recommendation order. The strongest candidate should appear first.

## Ranking Rules

Sort candidates by:

1. Strongest overall recommendation and role fit.
2. Ability to meet must-have requirements.
3. Assignment result when available.
4. Stable availability and earliest start date.
5. Soft skills: communication, initiative, ownership, resilience, empathy, structure, and repeated-outreach fit.
6. Motivation clarity and communication quality.
7. Lower risk level when candidates are otherwise similar.

Use `unknown` when evidence is missing. Do not guess hometown, grade, availability, start date, AI/tool usage, soft-skill performance, or assignment performance.
